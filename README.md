## Quick Start

```bash
# Clone the repository
git clone git@github.com:ls1intum/Apollon2.git
cd Apollon2

# Use correct Node.js version
nvm install && nvm use

# Install dependencies
npm install

# Build all packages
npm run build

# Start the project
npm run start
```
